\section{绪论}

\paragraph{数据和数据结构}
数据可以分为数值性数据,包括各种整型,浮点型,复数等数据类型;和非数值数据,如文字,图片,语音等.

\paragraph{数据结构的分类}
线性结构可以分为\emph{线性结构(linear structure)}和\emph{非线性结构(nonlinear structure)}.

\begin{itemize}

  \item 线性结构\\ 又称线性表,所有数据元素都按某种次序排列在一个序列中.这种结构又可以根据存储方式的不同,分为直接存取结构,顺序存取结构,字典结构.
  \begin{itemize}
    \item 直接存取\\ 访问元素可以直接访问,不必访问其前驱.
    \begin{itemize}
      \item 顺序表(数组)
    \end{itemize}
    \item 顺序存储\\
        访问元素时需要先访问其前驱.
    \begin{itemize}
      \item 表
      \item 栈
      \item 队列
      \item 优先队列
    \end{itemize}
    \item 字典结构
    \begin{itemize}
      \item 词典
      \item 散列表
    \end{itemize}
  \end{itemize}
  
  \item 非线性结构
  \begin{itemize}
    \item 层次聚类
    \begin{itemize}
      \item 树
      \item 堆
    \end{itemize}
    \item 群聚类
    \begin{itemize}
      \item 集合
      \item 图
    \end{itemize}
  \end{itemize}
\end{itemize}
