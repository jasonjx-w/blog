\section{简介}
valgrind是一个程序检查器，用于发现代码中的错误。
可以使用的工具模块：
\begin{itemize}
  \item cachegrind 用于查看缓存不命中
  \item callgrind 追踪每个函数调用次数和调用开销
  \item helgrind 发现程序中潜在的条件竞争
  \item lackey 用来创建工具，用于打印关于程序的统计数据
  \item massif 查看使用了多少堆内存
  \item memecheck 检查内存器
\end{itemize}
